# API BeYourChef 🍲🥗

Proyecto del Módulo 2 — Full Stack Dev Bootcamp.

## Qué es

BeYourChef es una API de recetas diseñada para que te conviertas en el chef de tu propio hogar. Esta plataforma permite a los usuarios buscar recetas, planificar minutas semanales para el día a día y optimizar sus recursos sugiriendo platos basados exclusivamente en los ingredientes que ya tienen disponibles en casa.

## Stack

- Node.js + Express
- MongoDB + Mongoose
- JWT para autenticación
